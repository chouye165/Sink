# 2026-07-14

## Task: Public short-link generator + dashboard source filter (Sink)

Changed the Sink link-shortener so the landing page is a public, no-login short link
generator, and the dashboard can filter links by who created them (console/admin vs guest).

### Changes

- `shared/schemas/link.ts`: added `source: z.enum(['admin','guest']).default('admin')` to `LinkSchema`.
- `server/utils/link-processing.ts`: `mergeEditableLink` now preserves `source` from `existingLink` (admin edits don't reattribute guest links).
- `server/utils/link-store.ts`: `listLinks` accepts optional `source` filter; loops KV pages until `limit` matches collected; normalizes legacy links (missing source -> 'admin').
- `server/api/link/list.get.ts`: accepts `source` query param, forwards to `listLinks`.
- `server/middleware/2.auth.ts`: bypass auth for `POST /api/link/guest` (sets `authMethod='guest'`).
- `server/api/link/guest.post.ts` (NEW): public, unauthenticated creation; restricted schema (url + optional slug/comment only; strips extra keys); marks `source:'guest'`; runs safe-browsing check via `prepareIncomingLink`.
- `app/pages/index.vue`: replaced marketing page with public generator form (URL + optional custom slug, randomize, copy result). i18n keys under `home.*`.
- `layers/dashboard/app/composables/links.ts`: added `sourceFilter` ('all'|'admin'|'guest') to `useDashboardLinksStore`.
- `layers/dashboard/app/components/dashboard/links/Index.vue`: passes `source` to list API; resets+reloads on filter change.
- `layers/dashboard/app/components/dashboard/links/SourceFilter.vue` (NEW): dropdown filter (All / Console / Guest).
- `layers/dashboard/app/pages/dashboard/links.vue`: added `<DashboardLinksSourceFilter />` to header.
- `layers/dashboard/app/components/dashboard/links/Link.vue`: shows a source Badge (Console/Guest) on each card.
- `shared/types/link.ts`: excluded `source` from `LinkFormFields` (admin editor must not send it).
- i18n: added `home.*`, `links.filter.*`, `links.source.*` to all 11 locale files (zh-CN/zh-TW translated; others English fallback).
